# Video Watch Time
### version 0.1.0
# Installation
  - `git clone github.com/flightman69/video_watchtime.rs`
  - `cargo build -r`
  - `cd video_watchtime` 
  - `cp target/release/videowatchtime $HOME/.cargo/bin/`

# Usage:
  - videowatchtime path
